# ponder

`ponder` is an early-stage open-source platform for computational chemistry.
The long-term goal is a polished desktop-first application and reusable modern
C++ platform for molecular data, workflows, visualization, plugins, scripting,
and local or remote computation.

The project is intentionally at the planning and scaffold stage. The first
milestone is not a chemistry engine; it is a durable foundation: repository
structure, CMake targets, tests, project loading, and clear subsystem
boundaries.

## Project Status

- Language: C++23.
- Build system: CMake.
- Initial repository model: monorepo.
- Initial target platforms: Windows, Linux, and macOS.
- License: Apache-2.0.
- Contributor certification: Developer Certificate of Origin.
