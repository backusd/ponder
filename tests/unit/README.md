# Unit Tests

Fast library-level tests live here.

Unit tests should avoid GPU, network, desktop window, and external chemistry
engine requirements.

## Initial Targets

- `ponder_core_tests`
- `ponder_math_tests`
- `ponder_chemistry_tests`
- `ponder_scientific_data_tests`
- `ponder_project_tests`
- `ponder_workflow_tests`
- `ponder_compute_tests`
- `ponder_io_tests`
- `ponder_platform_tests`
- `ponder_ui_tests`
- `ponder_plugin_sdk_tests`

`ponder_render_tests` is intentionally absent until REND-004 introduces
substantive public render behavior to test.
