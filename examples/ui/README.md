# UI Examples

These are manual contributor workbenches for the ponder UI rendering path.
They exercise public platform/render APIs plus explicitly experimental UI
facades; they are not a commitment to the future retained-mode API.

## Available Now

- [1-rectangle](1-rectangle/README.md): renders an ordered opaque/translucent
  rectangle scene, follows resize and suspension, and demonstrates explicit UI
  renderer teardown.

The UI boundary contract lives in
[libs/ui/docs/Boundary.md](../../libs/ui/docs/Boundary.md). The UI
implementation roadmap lives in
[libs/ui/docs/roadmap.txt](../../libs/ui/docs/roadmap.txt).
