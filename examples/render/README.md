# Render Examples

These are manual teaching programs for the public `ponder_render` API and its
integration with `ponder_platform`. They are intentionally small enough to use
as starting points for applications.

## Available Now

- [`1-clear-window`](1-clear-window/README.md): creates one compatible native
  window, clears and presents through the render library, responds to window
  state changes, and demonstrates explicit renderer-before-window teardown.

The render boundary contract lives in
[`libs/render/docs/Boundary.md`](../../libs/render/docs/Boundary.md). The render
implementation roadmap lives in
[`libs/render/docs/roadmap.txt`](../../libs/render/docs/roadmap.txt).
